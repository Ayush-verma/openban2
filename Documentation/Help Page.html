Help Page

Downloading Meter Reading Using UUID 
1. Enter the required uuid in the UUID Text Box.
2. Enter the file name you want to save the file with in the File Name text box.
3. Press ‘Enter To Save’ button.
Your correct CSV (based on the uuid) will be downloaded and saved in the Data Directory at the server.

Saving Possible Annotation Names
1. Press the ‘Save Annotation Names’ Button
2. Enter the desired Annotation Name in the Annotation Name text box.
3. Choose the correct Data Series for which you want to add the Annotation Name.
4. Press the ‘Enter To Save’ button.

Deleting Possible Annotation Names  
1. Press the ‘Delete Annotation Names’ Button.
2. Choose the correct Data Series for which you want to Delete the Annotation Name.
3. Choose the Annotation name you want to delete.
4. Press the ‘Enter To Delete’ button.



Loading Meter Reading for required Data Series
1. Select the Data Series for which you want to load the meter readings.
2. Enter the start date and time in the ‘Enter Start Date’ text box.
3. Enter the end date and time in the ‘Enter End Date’ text box.
4. Press the Submit button.


Saving Annotations for a Data Series

1. Load the Meter Reading for the required Data Series.
2. Select the range for which you want to save the Annotation simply by holding left mouse key and moving the cursor over the range.


3. A pop-up (Jquery Window) will appear.
4. Refine/Correct your range in the Start Date and End Date text boxes.
5. Select the required Annotation name checkbox and press Yes to Save.
6. You can press No to discard the selection (range).


7. You can select multiple Annotations to save together by repeating the above steps.
8. Enter the ‘Save Annotations’ button to permanently save the annotations to the server.

Showing Annotations for a Data Series

1. Choose the data series whose annotations you want to save from the ‘Select Data Series’ option list.  

2. Press the ‘Show Saved Annotations’ button. A pop up will be displayed on successful retrieval of the saved annotations.

3. Press the ‘Show’ button to view the saved annotations.

Deleting Annotations for a Data Series
1. Load the Meter Reading for the required Data Series.

2. Press the ‘Show Saved Annotations’ button to view the saved annotations.

3. Click on the annotation you wish to delete. (Just click anywhere on the plotband/colored region)
4. Confirm your decision in the pop-up window.
